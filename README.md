# 🎬 Agnes Video Studio

基于 **PySide6 (Qt6)** 的 AI 视频生成桌面应用，接入 [Agnes Video v2.0](https://agnes-ai.com/doc/agnes-video-v20) 模型。

支持文生视频、图生视频，异步任务轮询、实时进度、内置播放器预览。

---

## ✨ 功能特性

| 能力 | 说明 |
|------|------|
| 📝 **文生视频** | 输入提示词生成视频 |
| 🖼️ **图生视频** | 拖入首帧图片，生成动态视频 |
| 📊 **实时进度** | 异步轮询，显示排队→生成→完成百分比 |
| ▶️ **内置播放器** | 生成后直接播放，支持播放/暂停/静音 |
| 📐 **分辨率/比例** | 720p/480p/1080p × 16:9/9:16/1:1/4:3/3:4 |
| ⏱️ **时长选择** | 5 秒 / 10 秒 |
| 📚 **历史画廊** | SQLite 持久化，缩略图、搜索、收藏 |
| 📋 **提示词模板** | 内置 8 种风格（电影感/动漫/梦幻/赛博朋克…） |
| ⭐ **提示词收藏** | 收藏常用 prompt |
| 🔁 **自动重试** | 网络抖动自动重试（视频 API 容易遇到） |
| 🌙 **深色主题** | Catppuccin Mocha 配色 |

## 🚀 快速开始

双击 **`run.bat`**（自动创建虚拟环境并安装依赖），或手动：

```bash
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
.venv\Scripts\python video_gui.py
```

### 获取 API Key

前往 [Agnes AI](https://agnes-ai.com/) 注册，创建 API Key。

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Enter` | 生成视频 |
| `Space` | 播放/暂停 |

> 💡 视频生成通常需要 2–5 分钟，请耐心等待进度条到 100%。

## 📁 文件结构

```
AgnesVideo\
├── .venv\              # 虚拟环境
├── video_gui.py        # 主程序（PySide6 GUI）
├── video_client.py     # API 客户端（httpx，异步轮询）
├── video_store.py      # 历史存储（SQLite）+ 配置
├── requirements.txt    # 依赖清单
├── run.bat             # 一键启动
├── agnes.ico           # 应用图标
└── README.md
```

**用户数据**（视频、历史、配置）存放在 `%LOCALAPPDATA%\AgnesVideoStudio\`。

## 🔌 API 细节

| 项目 | 内容 |
|------|------|
| 提交任务 | `POST /v1/video/generations` |
| 轮询任务 | `GET /v1/video/generations/{task_id}` |
| 模型 | `agnes-video-v2.0` |
| 分辨率 | `720p` / `480p` / `1080p` |
| 比例 | `16:9` / `9:16` / `1:1` / `4:3` / `3:4` |
| 时长 | `5` / `10` 秒 |
| 任务状态 | `queued` → `IN_PROGRESS` → `SUCCESS` |
| 视频格式 | MP4 |

> 📌 缩略图抽取依赖 ffmpeg（可选）。未安装时显示占位图，不影响功能。
